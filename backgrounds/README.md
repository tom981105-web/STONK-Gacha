# 주식시장 배경화면 이미지 폴더

이 폴더에 배경화면 이미지를 넣으면, Home에서 착용 시 Battle 게임 화면에 적용됩니다.

## 파일명 규칙 (아이템 ID와 동일, 소문자)
- 확장자: `.webp`(권장) / `.jpg` / `.png` 중 하나 (webp→jpg→png 순으로 자동 탐색)
- 해상도: 1920×1080 (16:9) 권장, 한 장당 400KB 이하 권장
- 가운데 영역에 핵심 요소 배치(모바일에서 가장자리가 잘릴 수 있음)
